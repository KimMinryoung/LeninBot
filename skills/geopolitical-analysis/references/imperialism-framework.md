# 레닌 제국주의론 분석 프레임워크

## 핵심 명제 (『제국주의론』, 1917)

1. **독점 자본주의**: 자유경쟁 자본주의 → 독점 단계로 이행
2. **금융자본의 지배**: 산업자본 + 은행자본의 융합
3. **자본 수출**: 상품 수출보다 자본 수출이 지배적
4. **국제 독점 연합**: 세계를 분할하는 자본가 연합 형성
5. **열강에 의한 세계 분할**: 식민지·반식민지 쟁탈

## 21세기 적용

| 레닌 시대 | 현대 등가물 |
|-----------|------------|
| 식민지 영토 분할 | 에너지 자원·희토류 통제권 |
| 관세 장벽 | 제재·공급망 배제 |
| 포함 외교 | 군사기지·동맹 네트워크 |
| 금융자본 수출 | IMF 조건부 대출·달러 패권 |
